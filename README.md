# 🔐 Task 5 - Capstone Project & Incident Response

## 📌 Project Title
Web Application Penetration Testing using DVWA

## 🎯 Objective
To identify and exploit web application vulnerabilities in a controlled lab environment and implement mitigation strategies.

## 🛠️ Tools Used
- Kali Linux
- DVWA (Damn Vulnerable Web Application)
- Web Browser

## ⚙️ Methodology
- Reconnaissance
- Vulnerability Identification
- Exploitation
- Analysis
- Mitigation

## 💣 Attacks Performed

### 1. SQL Injection
- Bypassed authentication using SQL query manipulation
- Extracted user data

### 2. Cross-Site Scripting (XSS)
- Injected malicious JavaScript payload
- Executed script in browser

### 3. Cross-Site Request Forgery (CSRF)
- Changed user password without proper authorization

## 🛡️ Mitigation Techniques
- Prepared Statements
- Input Validation
- CSRF Tokens
- Content Security Policy (CSP)

## 🚨 Incident Response
- Detected vulnerabilities
- Analyzed attack behavior
- Implemented fixes
- Prevented future attacks

## 📊 Outcome
Successfully demonstrated real-world web vulnerabilities and security solutions.

## ⚠️ Disclaimer
This project was conducted in a controlled lab environment for educational purposes only.
